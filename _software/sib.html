Obsolete

Development of {{ page.title }} is frozen.

I developed this program to create simple, command line based installers for my programs, but my needs have changed and the program hasn't kept up. To update it would be (a) a lot of work and (b) pointless because the superb Inno Setup does it all so well.

I strongly discourage anyone from continuing to develop this program – it's had its day. I don't even know if it runs on modern versions of Windows! Having said that if you want to look at the code you can download a copy of the final version below.

There is no support available for this program. Please do not report any bugs or contact me about it.

Overview

SIBuilder (Small Installation program Builder) is an open source software installation program creator that creates stand alone command line based install programs.

Install programs are created using SIBuilder's Wizard style interface, as below.

For more information please download the program's help file or tutorial.

Download

The following downloads for SIBuilder are hosted on GoogleDrive:

  • The executable program: sibuilder-exe.zip.

    This zip file contains the program's self-generated installer (sib3inst.exe) and a read-me file (ReadMe.html). Unzip the file into a new folder. Installation instructions can be found in the read-me file.

  • The source code: sibuilder-src.zip.

    This download includes the source code of the final version of the program. Create a new folder to hold the source code and then unzip the file into it, preserving the directory structure.

  • Build notes: sibuilder-buildnotes.txt.

    This file explains how to compile the source code.

Some DelphiDabbler library projects are required to build 8 Queens from source. They are:

  • Version Information Component.
  • Shell Folders Unit.
  • Window State Components.
  • Clipboard Viewer Component.
  • Stream Extension Classes.
