This page is a park for anything that doesn't fit elsewhere on this website.

At present there's just the one item:

Info-ZIP zip.exe & unzip.exe

I use Info-ZIP's zip.exe in most of my build scripts to create zip files. The site I have been recommending to download the program doesn't appear to have been maintained for a while and I'm concerned that it might go away. So I've decided to host zip.exe and unzip.exe myself, just to be on the safe side.

A zip file, info-zip.zip, has been uploaded to my Google Drive. The file contains Windows builds of zip v2.32 & unzip v5.52, along with the required license. Clicking the following button will open a page on Google Drive that lists the contents of info-zip.zip. You will need to click the download button to get it.

Get zip & unzip from Google Drive

Yes, I realise storing a zip extractor in a .zip file is a bit recursive, but don't tell me you haven't got one already .
